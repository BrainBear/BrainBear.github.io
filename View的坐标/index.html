View的坐标

Posted on 2018-08-29

前言

View 有很多获取坐标的方法，但是其中含义各不一样，所以来总结一下。

left、right、top、bottom

获取的是 View 原始状态相对父 View 的坐标，对 View 进行平移等操作不会影响这四个值

translateX、translateY

translationX 表示当前 View 相对原始位置 x 方向移动的距离，如果 View 没有发生移动则值为 0

getX、getY

view.getX() = view.getLeft() + view.getTranslationX()

scrollX、scrollY

scorllX 表示 View 内容相对 View 显示区域 x 方向的偏移，和其他方法不一样，这个是内容的偏移。

getLocationInWindow

获取 View 在当前 Window 的坐标

getLocationOnScreen

获取 View 在当前屏幕 的坐标

getGlobalVisibleRect

获取 View 可见部分相对于屏幕的坐标

getLocalVisibleRect

获取 View 可见部分相对于 View左上角的坐标
