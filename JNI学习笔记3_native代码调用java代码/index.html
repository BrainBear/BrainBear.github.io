  • 获取到methodID
    要通过GetMethodID方法来获取到想要的方法id，GetMethodID方法第一个参数是指定的类，第二个参数是需要获取的方法名，第三个参数是方法参数，因为java支持方法重载，同名的方法可以有不同的参数列表或返回值，所以在native层中需要通过指定方法签名来确定具体要调用的是哪个方法。如果是获取静态方法的methodID就需要调用GetStaticMethodID方法。

  • 调用
    JNI中提供了一系列CallMethod方法，其中的type代表返回值，由于我们这个接口没有返回值，所以调用的是CallVoidMethod。
