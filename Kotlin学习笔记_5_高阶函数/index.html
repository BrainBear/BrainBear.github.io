高阶函数

什么是高阶函数？高阶函数就是将函数作为参数或者返回值的函数。Kotlin 中内置了很多好用的高阶函数，可以让你写出更加简洁的代码。

map

如果使用过 RxJava 的话，应该对 RxJava 的 map 操作符很熟悉，map 操作符可以说是最常用的操作符了，它的作用就是将 A 映射成 B ，假如我们需要对一个集合内部执行一系列的操作生成另外一个集合，就可以用到 map。

map 的逻辑就是取出集合中的每个数据，将该数据作用于传入的函数，然后将返回值放入到新的集合返回。我们看一下源码：

我们以 IntArray 的 map 为例，其他类型的数组也有 map 拓展函数，只是参数类型不一样。从上面的代码可以看到 IntArray 的 map 函数参数只有一个形参 transform ，transform 参数是 Int ，返回值是 R 的函数类型，而 map 函数返回的就是 R 类型的集合。

map 函数创建了一个和原本集合相同大小的新集合，然后把新集合和 transform 函数传给了 mapTo 函数。

mapTo 函数逻辑就是将原集合的数据遍历取出，然后传给 transform 执行操作，然后将返回值存到新集合，操作完之后返回新集合。

apply

apply 相当于增强版本的 let ：

apply 函数参数是一个函数类型的 block ，apply 内部就是执行了 block 函数，然后将自己返回。重点就是 block 函数，可以看到 block 是一个带接收者的函数，而这个接收者类型也和调用 apply 函数的是同一个类型，而带接收者类型的函数有一个特性：在函数内部有一个引用指向接收者类型。

上面可以清楚看到 it 和 apply 的区别，在 apply 中有引用指向调用者，所以可以省略 it 直接调用目标对象的方法和属性。
