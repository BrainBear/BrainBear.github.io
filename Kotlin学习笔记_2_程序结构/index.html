if && when

在 Java 中常用 if ，switch，三目运算符 ? 来控制代码分支，但是在 Kotlin 中 if 是一个表达式，它会返回一个值，因此 if 也可以实现 Java 中 ? 的效果：

在 Kotlin 中 if 分支可以是代码块，代码块最后的表达式作为该分支的值：

如果是分支比较多的时候，在 Java 中我们一般使用 switch ，但是在 Kotlin 中提供了增强版的 when 。

when 会将参数与所有分支条件按照顺序比较，直到某个分支满足条件，如果有多个条件执行同样的逻辑，可以使用 , 并列多个条件。when 既可以当做表达式也可以当成语句使用，如果 when 用于表达式，则必须要覆盖所有的条件，否则编译出错。

对于 Java 来说，switch 只支持特定的类型，而 when 支持任意表达式作为分支条件：
