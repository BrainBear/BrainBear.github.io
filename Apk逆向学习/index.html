Apk 逆向学习

首先说明一下我的环境，我使用的是 mac 环境，如果只是查看源码的话我记得 windows 平台有个更简单的软件可以用，但是我忘记叫什么名字了。。

工具

需要以下几个工具：

  • ApkTool 用于 Apk 的反编译和回编译
  • dex2jar 用于将 dex 文件转换成 class 文件
  • JD-GUI 用于查看 class 文件

下载好上面的工具，按文档配置好就可以了。
