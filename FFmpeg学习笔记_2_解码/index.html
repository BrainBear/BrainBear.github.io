打开输入

打开输入主要涉及了 AVFormatContext ，主要的属性如下：

FFMpeg 提供了两个方法来创建和释放 AVFormatContext :

我们需要通过 int avformat_open_input(AVFormatContext **ps, const char *url, AVInputFormat *fmt, AVDictionary **options); 方法来打开输入

  • AVFormatContext **ps AVFormatContext 指针的指针 可以传入一个指向 NULL 的 AVFormatContext 的指针，这样该方法内部会自动创建 AVFormatContext 并且赋值
  • const char *url 需要打开的输入流的 url
  • AVInputFormat *fmt 如果不为 NULL 则指定输入的格式，一般都传 NULL 让它自动检测
  • AVDictionary **options 一些额外的参数 在 FFmpeg 源码目录下 ./libavformat/options_table.h 文件有定义，一般传 NULL

一般情况下，调用 avformat_open_input 方法就可以在 AVFormatContext 获取到需要的信息，但是对于部分没有头部信息的格式来说，还需要调用 avformat_find_stream_info 方法来获取信息填充到 AVFormatContext 中：
