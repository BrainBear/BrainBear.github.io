Boolean

Char

Char 占 2 个字节，表示一个 16 位的 Unicode 字符，字符是用单引号 '' 括起来的， 例如'a', '0', '\n'。

装箱类型和基本类型

在 Java 中存在装箱类型和基本类型，例如 int ，和 Integer ，但是在 Kotlin 中只有 Int ，其实 Int 在 JVM 中是 int 和 Integer 的组合，由 Kotlin 编译器决定到底是使用基本类型还是装箱类型，程序员不需要关心也没办法控制。

类型转换

在 Java 中支持类型的隐式转换，例如把一个 int 类型的变量赋值给 long 类型。这是因为 long 类型表示的范围比 int 大，long 兼容 int。但是在 Kotlin 中，Int 类型并不能隐式转换为 Long 类型。基本类型之间的转换必须使用 Kotlin 提供的方法：

空类型安全

在 Kotlin 中任意类型，包括 Int ，String 和自定义的类型都有可空和不可空两种，默认的都是不可空类型，如果需要声明可空类型，需要在类型后面加 ? 。

区间

Kotlin 对比 Java 新增了现在流行的区间 Range 的概念，用于表示范围，最常用的是 IntRange。

定义一个闭区间需要使用 .. ， 例如 0..100 定义一个从 0 到 100 的闭区间 [0, 100]，如果你想要定义半开区间 [0, 100)，只能使用 until ，例如 0 until 100。

区间常用的方法有 isEmpty() 和 contains() ， 而且 Kotlin 还提供了 in 来简化 constans()。
